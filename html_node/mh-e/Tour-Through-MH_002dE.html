3 Tour Through MH-E

This chapter introduces some of the terms you'll need to know and then takes you on a tour of MH-E1. When you're done, you'll be able to send, read, and file mail, which is all that a lot of people ever do. But if you're the curious or adventurous type, read the rest of the manual to be able to use all the features of MH-E. I suggest you read this chapter first to get the big picture, and then you can read the manual as you wish.
