14.8.2.3 :file

An external :file that saves the results of execution of the code block. The :file is either a file name or two strings, where the first is the file name and the second is the description. A link to the file is inserted. It uses an Org mode style [[file:]] link (see Link format). Some languages, such as ‘R’, ‘dot’, ‘ditaa’, and ‘gnuplot’, automatically wrap the source code in additional boilerplate code. Such code wrapping helps recreate the output, especially graphics output, by executing just the :file contents.
